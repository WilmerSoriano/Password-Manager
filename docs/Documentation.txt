
Project: 
    Password Manager Application

Requirements:
    * Securely store and manage user passwords
    * User authentication with hashed passwords
    * Password encryption using OpenSSL
    * GUI built with Qt for cross-platform compatibility
    * Database backend using MySQL/MariaDB
    * Ability to add, edit, delete, and search password entries
    * Generate strong passwords
    * Backup and restore password database

Design:
    Language: C++ version 17 (https://en.cppreference.com/w/cpp/17)
    Library: OpenSSL version 3.0.13 (https://docs.openssl.org/3.0/man3/#copyright)
    Compiler: GCC version 13.3.0 (https://gcc.gnu.org/onlinedocs/gcc-13.3.0/gcc/)
    Builder: CMake version 3.28.3 (https://cmake.org/cmake/help/v3.28/index.html)
    GUI Framework: Qt GUI version 6.9.3 (https://doc.qt.io/qt-6/qtgui-index.html)
    Database: MySQL version 15.1 Distrib 10.4.32-MariaDB (https://dev.mysql.com/doc/refman/8.4/en/sql-data-definition-statements.html)
